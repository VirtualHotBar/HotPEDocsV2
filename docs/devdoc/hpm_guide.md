# HPM 介绍与指南

HPM（Hot Program Module）是一种模块打包格式，最初用于 HotPE 的模块化与模块商店生态。它适用于 PE 或桌面系统中的“免安装/低入侵”软件分发：打包、加载、卸载更直观，残留更少。

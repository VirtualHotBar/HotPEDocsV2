# 模块投稿

:::warning 命名要求（必读）
上传文件名请严格使用以下格式（建议用英文下划线 `_` 分隔）：

`软件名_打包者(你)_软件版本号_模块介绍.HPM`

示例：`AIDA64_VirtualHotBar_6.33.5700_全能的硬件检测工具.HPM`
:::

## 投稿前检查清单
- **避免重复**：同软件允许投稿新版本，但请先在 HotPE Client 模块商店搜索确认（名称/作者/版本）。
- **路径规范**：脚本和快捷方式中使用相对路径或环境变量，避免写死盘符与绝对路径。
- **可用性**：请先在 HotPE 环境中自测可正常运行（建议参考《[制作HotPE模块](/devdoc/makehpm)》完成打包与测试）。
- **安全合规**：不包含恶意行为、捆绑、隐私收集、破解/后门等内容。

上传后会进入审核流程；审核通过后会上架到模块商店。

如投稿 10 个以上模块，可加入[感谢名单](https://docs.hotpe.top/overview/thanks.html#%E6%A8%A1%E5%9D%97)！

## 审核结果
可使用下面的工具根据你投稿时的 **HPM 文件名** 查询当前审核状态：
<HpmAuditQuery />

如果上面工具不可用，则可去查看投稿链接，如你投稿的模块被移动到“已审核”文件夹，并且在 HotPE Client 中可搜索到，说明已审核通过；不能搜索到通常为未审核通过（请优先核对上面的投稿要求）。

## 解决方法

> 从上到下，逐个尝试

- 重启电脑后退出杀软重试。
- 换质量较高的U盘重试，不用水货杂牌。
- 去官网重新下载程序重试。
- 去官网下载旧版安装器进行安装。
- 下载ISO文件，通过其他工具写入 (如UltraISO)。
- 使用Ventoy制作U盘，然后下载ISO文件复制进U盘。

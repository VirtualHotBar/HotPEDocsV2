# 如何安装 HotPE 到 U 盘或系统

:::tip 安装前先下载资源
安装前请先在 HotPE Client 首页下载资源；如果未下载，点击安装时会有提示。未安装客户端可先参考《[如何下载 HotPE 及模块](/course/down)》。
:::

:::warning 重要：提前备份数据
向 U 盘/移动硬盘写入启动环境时，可能涉及分区/格式化等操作；请先确认目标设备并备份重要数据。
:::

## 安装到 U 盘或移动硬盘
1. 插入 U 盘或移动硬盘，打开 HotPE Client（建议以管理员身份运行）。
2. 点击侧边栏「安装到 U 盘」。

![](https://sc.sysri.cn/i/p/1/6991c1338fc60.png)

3. 选择要安装的设备，点击「开始安装」，等待完成。
4. 安装完成后，从 U 盘启动即可进入 HotPE（可参考《[如何进入 HotPE](/course/intohotpe)》）。

## 安装到系统（硬盘）
1. 打开 HotPE Client，点击「安装到系统」。
2. 按需进行个性化设置后，点击「开始安装」，等待完成。

![](https://sc.sysri.cn/i/p/1/6991c23cd94b1.png)

安装完成后，通常会在开机启动菜单中新增 HotPE 启动项，可在重启后选择进入（可参考《[如何进入 HotPE](/course/intohotpe)》）。

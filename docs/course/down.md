# 如何下载HotPE及模块
## 下载安装包
打开[HotPE官网](https://www.hotpe.top/)，在下载页面选择“立即下载”或通过其他方式下载。

![](https://sc.sysri.cn/i/p/1/6991bf229cf29.png)

## 下载模块
打开[HotPE官网](https://www.hotpe.top/)，在下载页面选择“下载模块”，在下载站中打开对应的模块分类文件夹进行下载

![](https://sc.sysri.cn/i/p/1/6991bfd135846.png)

其他下载方式:可通过HotPE Client模块管理或者PE中的模块商店下载

![](https://sc.sysri.cn/i/p/1/6991c270beca2.png)

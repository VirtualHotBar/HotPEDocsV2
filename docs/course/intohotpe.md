# 如何进入HotPE
## 从USB或光盘启动
启动方法是在开机时让BIOS从USB或光盘启动。有两种方式：
1. 设置BIOS从USB启动
2. 开机时按启动快捷键，选择需要启动的移动介质（推荐 因为启动较快）(由于制作启动U盘会格式化U盘，请备份好数据后再操作。)

## 从硬盘启动
每次开机时您都可以选择进入PE。PE并不是默认启动项，而是提供一个有默认3秒等待时间的启动项选择界面。您可以用方向键的上下键选择进入PE，如果不选择启动项就自动进入本地系统。
